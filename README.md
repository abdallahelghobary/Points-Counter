# Points-Counter

A cross-platform mobile application for tracking and managing points, built with modern mobile development technologies.

## 📱 About

Points-Counter is a versatile points tracking application designed to help users manage and monitor points across various activities and events. The application is developed using a multi-platform approach to ensure seamless performance on both iOS and Android devices.

## 🛠️ Technology Stack

### Primary Language
- **Dart** (90.1%) - Core application logic using Flutter framework

### Platform-Specific Code
- **Swift** (7.9%) - iOS native components and integrations
- **Kotlin** (1.6%) - Android native components and integrations
- **Objective-C** (0.4%) - Legacy iOS support and compatibility

## 🚀 Features

- **Cross-Platform Support**: Seamlessly runs on iOS and Android devices
- **Efficient Performance**: Optimized Dart code with native platform integrations
- **Modern UI/UX**: Built with Flutter for a responsive and intuitive interface
- **Points Tracking**: Easily track, add, and manage points in real-time

## 🔧 Development

### Prerequisites
- Flutter SDK
- Dart SDK
- Xcode (for iOS development)
- Android Studio (for Android development)

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/abdallahelghobary/Points-Counter.git
   cd Points-Counter
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the application**
   ```bash
   flutter run
   ```

### Project Structure

The project follows a standard Flutter application structure:
- Core Dart/Flutter code for business logic and UI
- Swift modules for iOS-specific features
- Kotlin modules for Android-specific features
